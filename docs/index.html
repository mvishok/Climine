Get Started

Welcome to Climine, a lightweight and straightforward scripting language designed for simplicity and customization. Climine aims to provide an easy-to-use environment for scripting tasks and learning the basics of programming logic

Installation

To get started with Climine, simply install it on your system. Get the latest installer from the releases page and run it. The installer will guide you through the installation process.

Running the First Script

Once installed, create a Climine script using your preferred text editor and run it using the Climine interpreter. To run a script, simply type the following command in the terminal:

                            
                            	$ climine    script.cli

For an interactive experience, enter the interactive mode to execute commands on the go. To run Climine in interactive mode, type the following command in the terminal:

                            
                                $ climine

Configuration

As of now, Climine only supports "logging" as a configuration option. To enable logging, pass the --log flag with the file name to log to. For example:

                        	
                            	$ climine --log log.txt

Options Value Default
log Logs the output to a file passed as an argument false

Syntax

Climine's syntax is designed to be clear and straightforward, making it accessible for both beginners and experienced developers. Here's an overview of the fundamental elements in Climine's basic syntax:

Variables

Variables are used to store values. To declare a variable, use the let keyword followed by the variable name and the value to be stored. For example: 

                        		
                                    let variableName = "Hello, Climine!";
                                    let number = 10;
                                    let array = [1, 2, 3, 4, 5];

Variables are dynamically typed, allowing you to assign different data types to the same variable.

Statements

Climine consist of statements, which are individual units of code that perform specific actions. Statements in Climine are typically terminated with a semicolon ;. 

                            
                                    display("This is a statement");

Expressions

Expressions are combinations of values, variables, and operators that result in a single value. Climine supports a range of operators for arithmetic, logical, and comparison operations. 

							
								let sum = (10 + 5);
								let isTrue = (15==15);

NOTE: Expressions with more than one term should be enclosed in parentheses.

Display Function

The display() function is used to output information to the console. It can be used to display strings, variables, and expressions. For example: 

							
								display("Hello, Climine!");
								display(10);
								display(10 + 5);

Input Function

To interact with the user, Climine provides the input function for receiving input. For example, the following code prompts the user to enter their name and displays a greeting message: 

                            
                                let name = input("Enter your name: ");
                                display("Namaskaram, " & name);

These are the foundational elements of Climine's basic syntax. As you explore more, you'll find that the simplicity of the syntax enhances readability and facilitates efficient script creation.

Control Flow

Climine provides intuitive constructs for controlling the flow of your scripts, allowing you to make decisions and repeat actions based on conditions.

Conditional Statements

Conditional statements are used to perform different actions based on different conditions. Climine supports if and else statements for conditional execution. For example, the following code prompts the user to enter a number and displays a message based on whether the number is positive or negative: 

                            
                                let number = input("Enter a number: ");
                                if (number > 0) {
                                    display("The number is positive");
                                } else {
                                    display("The number is negative");
                                };

Loops

Loops are used to repeat a block of code until a condition is met. Climine supports while and until loops for conditional execution. For example, the following code displays the numbers from 1 to 10: 

                            
                                let i = 1;
                                while (i < 11) {
                                    display(i);
                                    let i = (i + 1);
                                };

Climine also supports else statements for loops. The else statement is executed when the condition is no longer true.

NOTE: The until loop is the opposite of the while loop. It executes the block of code until the condition is met. Here's an example of an until loop: 

                            
                                let i = 1;
                                until (i > 10) {
                                    display(i);
                                    let i = (i + 1);
                                } else {
                                    display("The loop has ended");
                                };

Try-Handle-Finally

Climine supports the try, handle, and finally statements for exception handling. The try statement is used to test a block of code for errors. The handle statement is used to handle the error. The finally statement is used to execute code after the try and handle statements, regardless of the result. Here's an example of a try, handle, and finally statement: 

                            
                                try {
                                    let number = num(input("Enter a number: "));
                                    display("The number is " & number);
                                } handle {
                                    display("Given input is not a number");
                                } finally {
                                    display("The script has ended");
                                };

These are the foundational elements of Climine's control flow.

Functions

Functions are used to group a set of statements together to perform a specific task. Climine supports both built-in and user-defined functions.

Built-in Functions

Climine provides a set of built-in functions for performing common tasks. Here's a list of some of the built-in functions in Climine:

Function Description
append Append content to a file.
cmd Execute a shell command and return the output.
delete Delete a file.
display Display the given statement.
dump Display function definitions and variable values.
exists Check if a file exists.
input Take user input with an optional prompt.
num Convert a string to a numeric value.
now Get the current timestamp.
read Read the content of a file.
set Set the value of a variable.
split Split a string into an array based on a delimiter.
sum Calculate the sum of numeric values.
write Write content to a file, overwriting existing content.
exit Terminate the Climine program.

User-defined Functions

Climine also supports user-defined functions. To define a function, use the define keyword followed by the function name and the function body. For example, the following code defines a function named greet that takes a name as an argument and displays a greeting message: 

                            
                                define greet(name) {
                                    display("Namo", (name & "!"));
                                };

Calling Functions

To call a function, simply type the function name followed by the arguments in parentheses. For example, the following code calls the greet function defined above: 

                            
                                greet("Vasudevaya");

Returning Values

Functions can also return values. To return a value from a function, use the return keyword followed by the value to be returned. For example, the following code defines a function named sum that takes two numbers as arguments and returns their sum: 

                            
                                define sum(a, b) {
                                    return (a + b);
                                };

Calling Functions with Return Values

To call a function that returns a value, simply type the function name followed by the arguments in parentheses. For example, the following code calls the sum function defined above and displays the result: 

                            
                                display(sum(10, 5));

Scope and variables

Variables defined inside a function are local to the function. They cannot be accessed outside the function. Here's a brief example of how scope works in Climine: 

                            
                                define calculateSum(a, b) {
                                    let result = a + b;
                                    return result;
                                };
                                
                                let num1 = 5;
                                let num2 = 7;
                                let sum = calculateSum(num1, num2);
                                display("Sum:",sum);

The calculateSum function has its scope, and the result variable is local to the function. The num1, num2, and sum variables are global variables and can be accessed anywhere in the script.

Recursion

Climine supports recursion. A function can call itself to perform a task. Here's an example of a recursive function that calculates the factorial of a number: 

                            
                                define factorial(n) {
                                    if (n == 0) {
                                        return 1;
                                    } else {
                                        return (n * factorial(n - 1));
                                    };
                                };

The factorial function calls itself to calculate the factorial of a number.

External Integration

Climine allows you to integrate external scripts and programs into your script. You can execute shell commands and run external scripts from your script.

Including External Scripts

Climine allows you to include external scripts in your script. To include an external script, use the include keyword followed by the path to the script as a StringLiteral. For example, the following code includes the hello.cli script: 

                            
                                include "hello.cli";

Executing Shell Commands

Climine allows you to execute shell commands from your script. To execute a shell command, use the cmd function followed by the command to be executed. For example, the following code executes the ls command and displays the output: 

                            
                                display(cmd("dir"));
